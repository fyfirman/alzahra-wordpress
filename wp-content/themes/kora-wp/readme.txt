== Description ==
Kora WP is a feature rich Creative Portfolio WordPress theme which is ideal for Agencies, Creative, Freelancers, Photographers and other personnel involved in the design scene.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Upload your theme files kora-wp.zip
3. Click on the 'Activate' button to use your new theme right away.
4. Go to theme documentation for detailed instructions.
